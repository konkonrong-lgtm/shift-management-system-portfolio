## Project Overview

This project was planned, designed, developed, and operated to reduce the workload of shift submission and shift management in real store operations.

Main features include:

* Staff shift submission and correction
* Manager shift review
* Staff management
* Business hour and holiday settings
* Required staffing settings
* Shortage detection
* Additional recruitment message generation
* Excel export
* Owner / Manager / Staff role separation
* Auto Shift V1 draft generation
